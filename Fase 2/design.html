Propósito del sitio

Conocer todas las funcionalidades de las etiquetas y los atributos que nos puede presentar HTML, con el fin de que aprendamos a diseñar una pagina web desde cero, en donde podamos utilizar herramientas que nos simplifiquen el trabajo a la hora de realizar proyectos bajo la demanda de un cliente en especifico.






HEADER (esta etiqueta sirve para indicarle al navegador que todo lo que se almacene dentro de él, pertenece a la cabecera de nuestra página web).

NAV (esta etiqueta sirve para crear una lista, menú o panel de navegación).

SECTION (esta etiqueta sirve para crear una sección principal o almacenar información en una sección principal de nuestra página web o ingresar múltiple contenido relacionado con el contenido principal).

ARTICLE (esta etiqueta sirve para detallar el contenido importante de un artículo en su interior o para que ingresemos el contenido o información principal de nuestra página web, todo lo que se encuentre dentro de esta etiqueta, el navegador lo va a considerar IMPORTANTE).

ASIDE (esta etiqueta sirve para colocar información de forma lateral o contenido complementario relacionado con los artículos creados, pero que no es muy relevante, también se puede insertar citas de autores, hay que tener en cuenta de que los navegadores consideraran esta información "medianamente importante").

FOOTER (esta etiqueta sirve insertar pies de página como los son los copyright de las páginas web, a veces contienen enlaces y demás).

TITTLE (esta etiqueta nos permite colocarle un nombre que identifique de manera general a la pagina web).

META (con esta etiqueta le suministramos información de nuestra pagina web al navegador, para que esta sea tenida en cuenta cuando se genere una búsqueda que relacione la información que esta contenga).

CONTENT (este atributo nos permite agregar una descripción de nuestra pagina web o palabras claves que la describan y sean de utilidad para su ubicación).

LANG (este atributo nos permite designar el lenguaje en el se construyó la pagina web por ejemplo: lang="es" ... nos indica que es creada en español y que así lo buscara el navegador).
